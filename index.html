SynMob

A High-Fidelity Synthetic GPS Trajectory Dataset for Urban Mobility Analysis

Introduction

We present SynMob, a high-fidelity synthetic GPS trajectory dataset for urban mobility analysis. SynMob is generated by a novel diffusion probabilistic model, DiffTraj, which can generate trajectories with high fidelity and diversity. We help this Synthetic GPS Trajectory Dataset can facilitate the community for urban mobility analysis.
